Plataforma de Capacitación Gamificada – OXXO
Sistema full-stack desarrollado con socioformador OXXO para digitalizar la capacitación de asesores mediante un videojuego conectado a una plataforma web.

Desarrollado con C#, ASP.NET Core (Razor Pages) y Unity, integrando una API REST y base de datos relacional para almacenar métricas en tiempo real. Incluye autenticación de usuarios, dashboard con filtros dinámicos, leaderboard en vivo y foro con sistema de publicaciones y likes.

Tecnologías: C#, ASP.NET Core, Razor Pages, HTML, CSS, JavaScript, API REST, Unity, SQL.
